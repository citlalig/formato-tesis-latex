\chapter{Objetivos de la Investigación} 

\section{Objetivo General}
Desarrollar una aplicación que permita extraer los datos abiertos disponibles en una plataformas de datos abiertos de instituciones gubernamentales y que una vez obtenidos, los diferentes tipos de archivos que existen en diversos formatos sean estructurados uniformemente utilizando un formato único, de tal manera que puedan ser consultados y analizados de una manera más simple y natural.

\section{Objetivos Particulares}
\begin{enumerate}
\item Desarrollar un programa que inspeccione las páginas de una manera metódica y automatizada (Web Crawler en Inglés) para navegar a través del sitio de Datos Abiertos del Gobierno Mexicano (\texorpdfstring{\href{http://datos.gob.mx}
                {datos.gob.mx}}
                {datos.gob.mx})  y descargar la información requerida utilizando el API de CKAN mediante la ejecución de  un script en lenguaje Python.
\item Desarrollar un algoritmo para el procesamiento e indexado de la información que produzca como resultado archivos en formato JSON.
\item Proveer las capacidades de búsqueda de texto y definir un lenguaje de consulta utilizando las capacidades que la herramienta Apache Lucene proporciona. 
\end{enumerate}
